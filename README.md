# 2025_GURU2_Android
## MyMaps

안드로이드 Kotlin 기반 지도 앱으로, 장소 등록/조회/삭제 기능을 제공합니다.  
MVVM 패턴과 Room DB를 사용하여 구조적이고 유지보수 용이하게 설계되었습니다.

## 주요 기능

- 장소(Spot) 등록, 삭제, 조회
- 지도에서 장소 위치 확인
- 실시간 UI 업데이트 (LiveData & ViewModel)

---

## 사용 기술

- **언어:** Kotlin  
- **아키텍처:** MVVM  
- **DB:** Room (SQLite 기반 ORM)  
- **기타:** LiveData, ViewModel, RecyclerView
